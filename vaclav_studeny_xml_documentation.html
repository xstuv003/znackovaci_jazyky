Úvod

Moje semestrální práce se zabývá predevším XML souborem, kde je nastavení aplikace WarThunder. Nastavení aplikace Warthunder jsem si zvolil, jelikož mě dříve tato hra dost zaujala. Protože je hra velmi rozsáhlá, použil jsem její config nastavení a přepsal do XML souboru, který se potom využívá v této semestrální práci

Popis transformace

Závěr

Tato semestrální práce pro mě byla velmi zajímavá. Vyzkoušel jsem si poprvé práci s XML, který si myslím, že ještě v budoucnu použiji. Nastavení aplikace Warthunder je zdaleka větší a obsáhlejší, ale z důvodu rozsahů semestralní práce, bylo nastavení zkráceno.
